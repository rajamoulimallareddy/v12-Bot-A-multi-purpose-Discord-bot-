<h3> install</h3>
<li> install dependencies </li>
<pre> npm install </pre> 

<hr> 
<h3> Configration (config)  <h3>
<li> config.json </li> 
<pre>{ 
"token" : "Your Token here",
"prefix": ".",
"Owner": ["580381333914648579"],
"defaultPerms": ["SEND_MESSAGES"],

}</pre>
